An AI Agent Skill that **imitation-writes** viral **Xiaohongshu (小红书 / RED / RedNote / XHS)** notes — analyzing a reference post's structure, style, and traffic logic, then generating original, differentiated content that combines your product and selling points while bypassing duplicate detection.
